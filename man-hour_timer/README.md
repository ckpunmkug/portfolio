#### Таймер для измерения человеко-часов и фиксирования трудового процесса.

__О программе:__

Консольное приложение которое предназначено для измерения трудозатрат программиста или системного администратора, а так же для подтверждения выработки заявленных часов. Для фиксирования труда, таймер осуществляет ежеминутные screenshot, который затем может склеить в видеоролик с посекундной сменой screenshot.

__Доступный функционал:__
Задать время работы и время перерыва.
По Ctrl+C включается пауза.
Задать тип сигнала о завершении рабочего времени и перерыва.
Доступные сигналы: звуковой (beep), оповещение (notification).
Задать громкость звукового сигнала.
Задать папку для сохранения screenshot.
Задать workspaces для которых будет осуществляться screenshot.
Конвертировать папку с png в mp4 видеоролик.
Справка по практическому использованию ./start --help
