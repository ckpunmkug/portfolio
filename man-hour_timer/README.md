#### Таймер для измерения человеко-часов и фиксирования трудового процесса.

__О программе:__

Консольное приложение которое предназначено для измерения трудозатрат программиста или системного администратора, а так же для подтверждения выработки заявленных часов. Для фиксирования труда, таймер осуществляет ежеминутные screenshot, который затем может склеить в видеоролик с посекундной сменой screenshot.

__Доступный функционал:__

Задать время работы и время перерыва.<br />
По Ctrl+C включается пауза.<br />
Задать тип сигнала о завершении рабочего времени и перерыва.<br />
Доступные сигналы: звуковой (beep), оповещение (notification).<br />
Задать громкость звукового сигнала.<br />
Задать папку для сохранения screenshot.<br />
Задать workspaces для которых будет осуществляться screenshot.<br />
Конвертировать папку с png в mp4 видеоролик.<br />
Справка по практическому использованию ./start --help<br />
